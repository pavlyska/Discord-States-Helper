## 🌟 Описание

**Discord States Helper** — это минималистичный, но мощный бот, который предоставляет интерактивную систему помощи через команду `/dhelp`. Владелец может добавлять и удалять темы справки (например: *модерация*, *роли*, *правила*), а пользователи — выбирать интересующую тему и получать отформатированную информацию в виде embed'ов.

Всё хранится в `JSON`, без баз данных — легко развернуть и поддерживать.

---

## ✨ Особенности

✅ **Интерактивное меню выбора тем**  
✅ **Добавление тем через модальное окно (`/add`)**  
✅ **Удаление тем (`/remove`)**  
✅ **Поддержка изображений (через URL)**  
✅ **Автосохранение в `help_themes.json`**  
✅ **Защита команд — только владелец бота может редактировать**  
✅ **Отдельное меню для каждого пользователя (защита от чужих кликов)**  
✅ **Красивые embed'ы с цветами и иконками**

## 🔧 Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Установите зависимости:
   ```bash
   pip install discord.py
   ```

3. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-ник/Discord-States-Helper.git
   cd Discord-States-Helper
   ```

4. Откройте `main.py` и заполните данные:
   ```python
   BOT_TOKEN = "ВАШ_ТОКЕН_БОТА"
   OWNER_ID = 123456789012345678  # Ваш Discord ID
   GUILD_ID = 876543210987654321  # ID сервера (гильдии)
   ```

5. Запустите бота:
   ```bash
   python main.py
   ```

> ⚠️ Убедитесь, что бот имеет права:
> - `Send Messages`
> - `Use Application Commands`
> - `Manage Messages` (опционально, для чистки интерфейса)

## 🛠 Команды

| Команда | Описание | Кто может использовать |
|--------|---------|-----------------------|
| `/dhelp` | Показывает интерактивное меню помощи | Все пользователи |
| `/add`   | Открывает модальное окно для добавления новой темы | Только владелец бота |
| `/remove theme: [ключ]` | Удаляет тему по ключу | Только владелец бота |

## 🎨 Формат темы

Каждая тема сохраняется в JSON в таком формате:
```json
{
  "модерация": {
    "title": "Модерация сервера",
    "description": "На нашем сервере действуют следующие правила...",
    "color": 65536,
    "image": "https://i.imgur.com/example.png"
  }
}
```

> Цвет указывается в десятичном формате (можно использовать HEX → DEC конвертер).

---

## 🛡 Требования безопасности

- Команды `/add` и `/remove` защищены через `@app_commands.check(is_owner)`
- Меню выбора тем доступно только тому, кто вызвал `/dhelp`
- Ключи тем не должны содержать пробелы (проверяются при добавлении)
